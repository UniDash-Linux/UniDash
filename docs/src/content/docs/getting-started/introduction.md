# Welcome to UniDash

UniDash is a unified dashboard for managing self-hosted applications on Kubernetes.

## Features

- Web-based desktop interface
- Centralized authentication (SSO, AD, OIDC)
- Application store with one-click installation
- Multi-window management with tiling support
- Role-based access control
