## Key Features

<CardGrid stagger>
    <Card title="Web Desktop" icon="laptop">
        A familiar desktop experience in your browser with windows, dock, and
        tiling layouts. Works on desktop, tablet, and mobile.
    </Card>
    <Card title="Single Sign-On" icon="approve-check-circle">
        One login for all your apps. Supports OIDC, LDAP, and proxy
        authentication for seamless access.
    </Card>
    <Card title="App Store" icon="puzzle">
        Install self-hosted applications with one click. No YAML, no CLI, just
        point and click.
    </Card>
    <Card title="K3S Cluster" icon="rocket">
        Built on Kubernetes for high availability. Scale from a single node to a
        full cluster.
    </Card>
</CardGrid>

## Quick Install

```bash
curl -sL unidash.io/install | bash
```

Then open the displayed URL in your browser to complete the setup wizard.
